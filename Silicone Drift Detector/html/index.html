gccDppConsole

gccDppConsole is a console application that demonstrates how Amptek DPP communications can be easily integrated into other programs.

What gccDppConsole does.

gccDppConsole communicates to a DPP device and demonstrates a minimal set of acquisition related commands.

gccDppConsole Classes

  • CConsoleHelper (Simplifies function calls.)
  • CDppLibUsb (Windows LibUsb communication class.)
  • CDP5Status (DPP status processing.)
  • CParsePacket (DPP packet parsing.)
  • CDP5Protocol (Defines and implements DPP protocol.)
  • CSendCommand (Generates command packet to be sent.)
